## 题目描述

给定两个整数n和m，求[n, m]范围内所有素数的和。

## 代码

```c
#include<stdio.h>
#include<math.h>
int main()
{
    int n, m;
    scanf("%d %d", &n, &m);
    int i, j;
    long long sum = 0;
    for (i = n;i <= m;i++)
    {
        int x = 0;
        for (j = 2; j <= sqrt(i); j++) {
            if (i % j == 0)
            {
                x = 1;
            }
        }
        if (x == 0)
        {
            sum += i;
        }
    }
    printf("sum=%lld", sum);
}
```
