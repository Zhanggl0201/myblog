## 题目描述

使用二维数组打印杨辉三角。

## 代码

```c
#include<stdio.h>
#define N 30

int main( )
{
    int a[N][N];
    int n,i,j;

	for(i=0;i<n+1;i++)
    {
        a[i][0]=1;
        for(j=1;j<i;j++)
        {
            a[i][j]=a[i-1][j-1]+a[i-1][j];
            printf("%d ",a[i][j]);
        }
        printf("\n");
    }

    for(i=0;i<n;i++)
    {
        for(j=0;j<=i;j++)
        {
            printf("%d ",a[i][j]);
        }
        printf("\n");
    }

    return 0;
}
```
