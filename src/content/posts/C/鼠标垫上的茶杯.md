## 题目描述

作者闲来无事，泡了杯茶放在桌子的鼠标垫上，又拿来n张大小不一的鼠标垫，编号1-n，现在将这些鼠标垫依次铺好，每个鼠标垫按照顺序依次叠加起来，铺完n张鼠标垫之后，作者把茶杯放在了坐标(x, y)上，请问这杯茶现在在哪一张鼠标垫上？

**输入格式**：第一行为鼠标垫的数量n（0 < n <= 30）；接下来的n行，每行有四个参数，分别为鼠标垫摆放的坐标以及其在x轴上的长度及在y轴上的长度，均为整数，最后一行为茶杯坐标。

**输出格式**：输出杯子所在鼠标垫的编号（不在任何鼠标垫上输出-1）

## 代码

```c
#include <stdio.h>

int main(void) {
    int a[30][4];
    int n;
    int m[2];
    scanf("%d", &n);
    for (int i = 0;i < n;i++)
    {
        scanf("%d %d %d %d", &a[i][0], &a[i][1], &a[i][2], &a[i][3]);
    }
    scanf("%d %d", &m[0], &m[1]);
    for (int i = n - 1;i >= 0;i--)
    {
        if (((a[i][2] + a[i][0]) >= m[0] && (a[i][0]) <= m[0]) && ((a[i][3] + a[i][1]) >= m[1] && (a[i][1]) <= m[1]))
        {
            printf("%d", i + 1);
            return 0;
        }
    }
    printf("-1");
    return 0;
}
```
