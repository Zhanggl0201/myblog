## 题目描述

输入10个整数到一维数组中，计算非负数的和，并找出最大值和最小值。

## 代码

```c
#include <stdio.h>
#define N 10
int main()
{
    int a[N] = { 0 }, sum = 0, max = 0, min = 0;
    for (int i = 0;i < N;i++)
    {
        scanf("%d", &a[i]);
        if (i == 0)
        {
            max = a[i];
            min = a[i];
        }
        if (a[i] >= 0)
        {
            sum += a[i];
            if (a[i] > max)
            {
                max = a[i];
            }
            if (a[i] < min)
            {
                min = a[i];
            }
        }
    }
    printf("sum=%d\nmax=%d\nmin=%d", sum, max, min);

}
```
