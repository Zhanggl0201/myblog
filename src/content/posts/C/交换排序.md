## 题目描述

使用交换排序（冒泡排序）对数组进行升序排序。

## 代码

```c
#include <stdio.h>

int main() {
    int a[10] = { 2,4,2,57,64,23,45,67,89,12 };
    int i, j, flag;
    for (i = 0;i < 9;i++)
    {
        for (j = 0;j < 9 - i;j++)
        {
            if (a[j] > a[j + 1])
            {
                flag = a[j];
                a[j] = a[j + 1];
                a[j + 1] = flag;
            }
        }
    }
    for (i = 0;i < 10;i++)
        printf("%d ", a[i]);
    return 0;
}
```
