# 1.成绩等级判定
## 输入一个0-100之间的整数表示成绩，按规则输出等级：≥90为A，80-89为B，70-79为C，60-69为D，<60为E。若输入不在范围内，输出“无效”。
```python
score = int(input())
# 请在此编写代码
if 90<=score<=100:
    print('A')
elif 80<=score<=89:
    print('B')
elif 70<=score<=79:
    print('C')
elif 60<=score<=69:
    print('D')
elif 0<=score<60:
    print('E')
else:
    print('无效')

```

# 3.打印图形-沙漏
## 编写一个程序，根据用户输入的参数n，生成并打印对称的沙漏图形。沙漏图形由 “上半部分（字符数递减）+ 中间单行（1 个字符）+ 下半部分（字符数递增）” 组成，要求图形左右对称，填充字符统一，整体呈现沙漏形状。输入的参数n代表沙漏的最大宽度（即最顶部和最底部一行的字符个数），若输入偶数，程序自动修正为比其大 1 的奇数。
```python
def print_hourglass(height):
    for i in range(height):
        print(' ' * i + '* ' * (height - i))
    for i in range(height - 2, -1, -1):
        print(' ' * i + '* ' * (height - i))

# 示例
n=eval(input())
print_hourglass(n)
```

# 4.和亲数
## 毕达哥拉斯曾说：“朋友是你灵魂的倩影，要像 220 与 284 一样亲密”。 人和人之间存在友情，有趣的是，数与数之间也有类似的关系，数学家把一对存在 特殊关系的数称为“亲和数”，其定义如下： 
### 对于两个正整数 a 和 b，若同时满足如下条件，则称 a、b 是一对亲和数： 
###  a 除本身外的所有因数之和等于 b； 
###  b 除本身外的所有因数之和等于 a。 
### 注：因数也叫约数，若整数 x 除以整数 y 的余数是 0，则称 y 是 x 的因数，例如 284 的所有因数为：1、2、4、71、142、284（本题只考虑正因数）。 
### 例如，220 和 284 是一对亲和数，因为： 
###  220 的所有因数（除本身外）之和是 1+2+4+5+10+11+20+22+44+55+110=284； 
###  284 的所有因数（除本身外）之和是 1+2+4+71+142=220。 
### 请补全函数 factorsSum(N)和 isAmicable(a, b)。说明如下： 
### 1）函数 factorsSum(N)的功能是求正整数 N 除本身外的所有因数之和，例如 N 为 284 时，返回值 s 为 220； 
### 2）函数 isAmicable(a, b)的功能是判断正整数 a 和 b 是否为亲和数，返回值为布尔型， 如果是亲和数则返回 True，否则返回 False。
